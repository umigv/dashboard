# Robot Dashboard

Frontend interface to the UMARV robot.

## Runtime Dependencies
* Node.js LTS
* `yarn`
* The following, which can be installed with `yarn`:
    * `rclnodejs`
    * `express`
    * `socket.io`

## Development Dependencies
* The following, which can be installed with `yarn`:
    * `typescript`
    * `@types/express`

## Run
To run the dashboard, use `npm start`
